Carefully read the given scenario:

SkyRoute Airlines wants to digitize their operations. The airline operates multiple flights, each of which has a unique flight number (a 4-character alphanumeric code), a source city, a destination city, and a departure time. Every flight belongs to one of three categories: Domestic, International, or Charter. Each flight carries a number of passengers. A passenger has a unique passenger ID, a name, and a ticket class (Economy, Business, or First Class). The airline employs crew members who are assigned to flights. Each crew member has a unique staff ID, a name, and is paid based on flight hours logged. The airline must generate a revenue report for each flight category, though the calculation method differs per category.

Perform the following tasks:

  1. Identify and create all classes (with proper relationships) present in the scenario. Add all required variables and arrays with appropriate data types. Do not add constructors or functions yet.
  2. For classes created in part (a), write all default and parameterized constructors. If there are dynamic data members, initialize them properly using user-provided values.
  3. Add a mechanism such that whenever a flight is cancelled, all passengers booked on that flight are removed and lose any connection to the airline. Similarly, if a flight category is discontinued, all flights under it must be removed.
  4. Add a function ViewTotalPayroll() in the Flight class that computes and returns the total salary paid to all crew members currently assigned to that flight.
  5. Implement a Revenue Report mechanism for each flight category:
    • Domestic: Revenue = sum of all ticket prices minus 10% operational cost deduction.
    • International: Revenue = sum of all ticket prices plus a fixed international surcharge of PKR 5000 per passenger.
    • Charter: Revenue = a flat rate per flight (stored in the class) minus crew salaries paid.

Consider the given code and answer the following questions:

class Vehicle {
public:
    void start(int speed) { }
};

class Car : public Vehicle {
public:
    void start() { }
};

class Truck : public Vehicle {
    int load;
public:
    void start(int speed) { }
};

class Bike : public Vehicle {
public:
    void start(int speed) { }
};

class HybridVan :
    public Car, protected Truck,
    public Bike {
public:
    int load;
};

int main() {
    Vehicle v;
    v.start(60);

    HybridVan hv;
    hv.start(60);
}
  1. Does the given code suffer from the Diamond Problem? Provide a clear justification.
  2. If the Diamond Problem is present, modify the code to resolve it.
